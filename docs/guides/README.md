## 🤔 常见问题

### Q: 我应该选择哪个图床？

**新手推荐**：
```
TCL + 京东
理由：零配置，开箱即用
```

**日常使用**：
```
TCL + 京东 + 微博
理由：多种链接格式，访问稳定
```

**重要图片**：
```
TCL + 京东 + R2
理由：三重保障，长期稳定
```

**完整配置**：
```
微博 + R2 + TCL + 京东 + 牛客
理由：最大冗余，适合专业用户
```

### Q: 配置难度排序？

从易到难：
1. ⭐ **TCL / 京东** - 零配置
2. ⭐⭐⭐ **微博 / 牛客** - 需要 Cookie（10分钟）
3. ⭐⭐⭐⭐ **R2** - 需要详细配置（30分钟）

### Q: 哪个图床最稳定？

稳定性排序：
1. 🥇 **R2** - 商业级稳定性
2. 🥈 **京东** - 大厂支持
3. 🥉 **TCL** - 稳定可靠
4. **微博 / 牛客** - 需定期更新 Cookie

### Q: 哪个图床最快？

速度排序：
1. 🥇 **京东** - 速度最快
2. 🥈 **TCL** - 速度很快
3. 🥉 **R2** - 速度较快
4. **微博 / 牛客** - 速度一般

### Q: 所有图床都需要配置吗？

**不需要！**
- TCL 和京东**开箱即用**，无需任何配置
- 其他图床根据需求选择配置

### Q: Cookie 过期怎么办？

微博和牛客的 Cookie 会过期：
- 微博：约 30 天
- 牛客：约 7-30 天

**解决方法**：按照对应指南重新获取 Cookie 即可
