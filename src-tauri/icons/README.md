# 图标文件说明

此目录需要包含以下图标文件：

- `icon.png` - 系统托盘图标 (32x32 或 64x64 像素)
- `32x32.png` - 32x32 像素应用图标
- `128x128.png` - 128x128 像素应用图标
- `128x128@2x.png` - 256x256 像素应用图标 (用于 Retina 显示)
- `icon.icns` - macOS 图标文件
- `icon.ico` - Windows 图标文件

## 快速开始

如果您暂时没有图标，可以使用以下方法：

1. **使用在线工具生成图标**：
   - 访问 https://www.favicon-generator.org/ 或类似工具
   - 上传一个 512x512 的 PNG 图片
   - 下载生成的图标文件

2. **使用占位图标**：
   - 创建一个简单的 32x32 PNG 文件作为临时图标
   - 复制到所有需要的文件名

3. **从 weibo-picture-store 项目复制**：
   - 如果该项目有图标文件，可以直接复制使用

## 注意事项

- 图标文件必须存在，否则应用可能无法正常启动
- 系统托盘图标建议使用透明背景
- macOS 上建议使用 `.icns` 格式以获得最佳效果
